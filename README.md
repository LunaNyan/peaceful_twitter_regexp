# peaceful twitter regexp
평화로운 트위터 타임라인을 위한 정규 표현식

## All-in-One

```
((칼|배)박이|(도검|)난무|(깡|칸)코레|(좌|우)익|일[0-9]*(베|배)|메[0-9]*(갤|갈)|(김|씹)치(남|녀)|씹치(남|여)|몰카|소라넷|한남충|삼일한|(남|여)혐|키분|[kK][iI][bB][uU][nN]).*
```
